# Assets Folder

Place your Unfiltered Ventures logo here.

**Logo file name:** `unfiltered-ventures-logo.png`

**Recommended specifications:**
- Format: PNG (with transparency) or SVG
- Size: At least 200px width (height will scale proportionally)
- Background: Transparent or white
- The logo will display at approximately 48px height in the footer

If you don't have the logo file yet, the footer will fall back to showing "Unfiltered Ventures" as text.
